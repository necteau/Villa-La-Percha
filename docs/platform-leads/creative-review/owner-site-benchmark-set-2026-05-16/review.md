## Benchmark 1: WILDPOD

### Source Read

Adult-couples luxury glamping on the edge of downtown Tofino. Six geodesic waterfront domes with 5-meter panoramic windows, cedar decks, queen beds, kitchenettes, propane fireplaces, local toiletries/coffee, no TV, no children/pets, no smoking/open flames, quiet hours, and a 2-night minimum. The strongest source differentiator is the collision of wild Clayoquot Sound views with walkable town convenience.

### Three Directions

| Direction | Score / 50 | Notes |
|---|---:|---|
| Panoramic Window Romance | 45 | Best emotional hook: make the dome window the whole first-screen idea, then support it with adult-only quiet, fireplace, deck chairs, and no-TV positioning. |
| Tofino Micro-Basecamp | 42 | Strong for planning: restaurants, kayaking, seaplanes, shops, harbour activity, town access. Slightly less distinctive visually. |
| Eco-Luxury Minimal Pod | 43 | Good fit for no AC/no TV/Ocean Friendly constraints, but risks sounding like an operations memo if not warmed with guest desire. |

Selected base: **Panoramic Window Romance**.

Synthesis borrow list:

- Borrow town-access and activity planning from Tofino Micro-Basecamp.
- Borrow sustainability/no-TV/no-AC clarity from Eco-Luxury Minimal Pod.
- Keep the panoramic window, harbour weather, cedar deck, fireplace, and adult quiet as the emotional spine.

Final synthesized direction: **Clayoquot Window Stay**.

Final target score: **47/50**.

Preview Build lesson: when a property has one unmistakable architectural/experiential feature, the hero should be built around that specific object/experience rather than a generic destination promise.

## Cross-Property Lessons

1. **Object-first heroes beat category heroes.** WILDPOD should lead with the 5-meter panoramic window; Nkanyi with the after-safari pool/boma ritual; The Shippen with modern barn conversion + rural quiet. Broad labels like glamping, safari villa, or countryside cottage are weaker.
2. **Image classification matters more than image count.** Text cards, logos, feature graphics, collages, and OTA/promotional banners are source notes, not section art.
3. **Policy constraints can improve trust if framed correctly.** Adult-only, no pets, no TV, no AC, no smoking, quiet hours, and minimum stay rules should become confidence modules, not apology copy.
4. **Sparse owner-site copy is a real blocker.** Good photos do not excuse unsupported local claims. If extractable copy is thin, the Preview Build should expose owner questions rather than inventing nearby experiences.
5. **The Creative Review winner should synthesize planning clarity from lower-scoring directions.** The best visual concept often needs the logistics/planning direction's date, rule, local, and trip-confidence modules to become a serious direct-booking site.

## Recommended System Updates

- Add a playbook reminder that source media must be classified by role: hero candidate, early-section candidate, source-note-only, rejected.
- Treat text-overlay feature cards as structured fact-register inputs, never as guest-page imagery.
- Add an owner-detail request when image gate passes but source copy is thin.
- Encourage object/ritual-first hero concepts during Creative Review instead of broad property-category hero lines.

## Owner-Share Blockers

All three benchmark concepts remain internal-only until:

- photo rights / owner permission are confirmed;
- source facts are owner-confirmed or cited;
- rates, fees, taxes, minimum stay, cancellation, and operating rules are confirmed;
- local recommendations/distances are verified;
- desktop/mobile rendered QA exists against an actual DirectStay preview URL;
- Jaimal approves any external owner communication.
