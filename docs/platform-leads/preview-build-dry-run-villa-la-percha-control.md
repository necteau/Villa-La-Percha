# DirectStay Preview Build Dry Run — Villa La Percha Control

Status: process dry run only. This is stored under DirectStay PlatformLead docs, not Villa La Percha production docs. Villa La Percha is used here only as a known control property to test the Preview Build process.

## Goal

Test whether the Preview Build process produces a property-specific design brief from listing/site evidence before implementation.

## Source material used

Known DirectStay/Villa La Percha context from workspace memory and existing site work:

- Property: Villa La Percha
- Location: Chalk Sound, Providenciales, Turks and Caicos
- Property type: private villa / island retreat
- Guest promise: private Chalk Sound stay with water views, pool, calm island planning support, and direct-owner confidence
- Existing visual cues: turquoise water, white/cream architecture, pool blues, sunlit terraces, tropical greenery, warm neutral interiors

## Observed facts

- Villa La Percha is a private villa in Chalk Sound, Providenciales.
- It is positioned as a direct-booking alternative to OTA dependence.
- Guest experience content and island guide/map have been intentionally built out.
- Direct rates are tax-inclusive on DirectStay.
- The property benefits from strong visual/environmental cues: Chalk Sound water, pool, villa privacy, island planning.

## Inferred assumptions for design test

- Target guests are likely families/groups seeking privacy, confidence, and a polished island stay.
- Best emotional promise is not “book a rental” but “own a quiet piece of Chalk Sound for the week.”
- Design should feel elevated but not cold; island-luxury, not corporate travel.
- DirectStay value should emphasize confidence, local guidance, and owner relationship more than discount language.

## Property Design Fingerprint

- **Property type:** Luxury private villa / island retreat.
- **Guest archetype:** Family/group travelers, privacy-seeking island guests, direct-booking guests who want confidence before inquiry.
- **Emotional promise:** Calm, private Chalk Sound base with the clarity and care missing from marketplace listings.
- **Location cues:** Turquoise lagoon, limestone/sand, sunlit white architecture, tropical green, golden sunset.
- **Decor/style read:** Coastal refined / resort-adjacent; likely airy, clean, warm neutral, water-facing. Avoid heavy rustic or urban editorial styling.
- **Signature stay moments:** Pool and terrace time, Chalk Sound scenery, privacy, island planning/local guide confidence, direct relationship.

## Palette direction

Derived from property/location cues:

- Deep lagoon: `#0E6F78`
- Chalk Sound turquoise: `#62C7C7`
- Warm limestone: `#F4EBDD`
- Sunlit cream: `#FFF9EF`
- Palm green: `#5F7F61`
- Sunset coral accent: `#D98263`
- Ink/navy text: `#12313A`

Avoid:

- Generic SaaS blue.
- Over-saturated Caribbean postcard colors.
- Heavy black luxury that fights the airy island context.

## Typography / layout direction

- Display: elegant serif or high-contrast editorial display for hero/emotional headings.
- Body: warm, readable sans.
- Layout: full-bleed scenic hero, airy editorial image sections, soft owner-view callouts, rounded but restrained cards.
- Rhythm: spacious and sunlit; large image moments interleaved with practical confidence blocks.

## Chosen archetype

**Luxury villa / resort alternative** with island editorial treatment.

Why:

- The property competes with high-value OTA villa listings.
- The site should elevate the property into a standalone hospitality brand.
- Local guide content and guest confidence are strategic differentiators.

## Process lessons

- The fingerprint approach works: it quickly creates a property-specific design direction.
- The weakest current step is fresh image/decor analysis; for real leads, we should inspect listing images directly and extract palette/style from the photos before finalizing design.
- The preview should always include an “assumptions we will replace with owner truth” section.
- DirectStay needs 4–6 archetype seed DESIGN.md files next so implementation can be fast without becoming generic.
