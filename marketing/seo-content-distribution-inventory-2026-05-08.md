### 2. Travel planner / concierge value-add

Use the content as helpful planning collateral rather than a cold sales blast:

- Primary link: property page.
- Support link: itinerary builder or island guide.
- Framing: “This is a direct-booking villa page plus planning guide you can use to quickly evaluate whether Chalk Sound fits a family/private group.”
- Approval-needed artifact: concise planner email with 2 links max.

### 3. Destination-group/community posts

Only where rules allow helpful destination advice:

- Approval-needed artifact: one group-safe post and one comment template, stripped of hype.

### 4. Network announcement support links

For personal/family/friend sharing:

- Use the property page as the main CTA.

- Keep direct-booking savings language conservative and tied to current approved copy.
